# Unhidden

### Overview

Unhidden is a simple app for macOS to show hidden files and folders in macOS System.

### How does it work?

Well, simple! Launch the app, toggle the switch. All finder windows, if any open, will automatically relaunch. If that does not happen, hold `‚å• (Option)`, `^ (Control)` click **Finder** then select **Relaunch**.
